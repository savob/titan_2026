# Microcontroller Code

This folder contains the firmware projects for the embedded system within TITAN going into competition for 2026, targeting the STM32 microcontroller on board. The base for these is the original project from 2022, written using the Arduino IDE within the PlatformIO system.

All code going into the embedded system was written in C/C++.

## TITAN PlatformIO Code

This is an updated version of the code from 2022 for the central microcontroller in the system, I do not expect there to be any major changes in this code as it worked well for us then. The main updates will be to reflect some hardware changes in the connections and some sensors used for capturing information.

As mentioned earlier this is written in C/C++ using the Arduino framework within the PlatformIO system for VS Code.

# TITAN Hardware

These folders contain the projects related to hardware designed to operate TITAN.

They were all designed using KiCad 10.0.

## TITAN Board

The main TITAN hardware is designed to be a HAT (**HA**rdware on **T**op) for the Raspberry Pi 3s we use for video feeds. These HATs perform power regulation for their video systems, data collection, and orchestrate communications between the different systems both inside and outside of TITAN.

## Wheel Board

In TITAN there is small board designed to be located near the wheel brake disks to monitor wheel speed and brake disk temperature for the main board to monitor.
